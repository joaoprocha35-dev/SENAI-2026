# 📱 App Mobile - Estudos (Projeto SENAI)

Aplicação mobile desenvolvida com **React Native** e **Expo**, focada na experiência do usuário e integração em tempo real com uma API Backend.

## 🚀 Tecnologias

* **React Native** (Expo)
* **JavaScript**
* **Axios** (Consumo de API REST)

## 🔗 Integração

Este aplicativo é o Front-end do projeto. Ele consome os dados e serviços da API desenvolvida em Node.js, que você pode encontrar no link abaixo:
👉 [https://github.com/joaoprocha35-dev/api-mobile-senai](https://github.com/joaoprocha35-dev/api-mobile-senai)

## 📂 Funcionalidades

* **Gerenciamento de Perfil:** Visualização e edição de dados do usuário através dos campos de nome e perfil.
* **Consumo de API:** Integração completa com o servidor para persistência de dados.
* **Navegação Dinâmica:** Fluxo otimizado entre as telas do aplicativo.

## ▶️ Como rodar o projeto**

Siga os passos abaixo no seu terminal para executar o app:

1. Instale as dependências do projeto:
```bash
npm install

2. Inicie o servidor do Expo:
npx expo start
