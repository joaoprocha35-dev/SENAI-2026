# App_DOM 🏗️

Nesta pasta estão concentrados os projetos práticos focados em **Manipulação de DOM** (Document Object Model). O objetivo é demonstrar a aplicação de lógica JavaScript para criar interfaces interativas e dinâmicas.

## 📁 Projetos Inclusos:

* **To-Do List Gamificado**: Gestão de tarefas com atribuição de pontos, cálculo de saldo em tempo real e remoção dinâmica de itens.
* **Calculadora**: Lógica de operações matemáticas e atualização de display.
* **Contador com Limite**: Controle de estados numéricos com travas de segurança.
* **Relógio Digital**: Sincronização de tempo real utilizando `setInterval`.

## 🛠️ Conceitos Aplicados:

* **Seleção de Elementos**: Uso de `getElementById` e `querySelector`.
* **Gestão de Eventos**: `addEventListener` para cliques e interações.
* **Manipulação de Atributos**: Uso de `dataset` para armazenar valores (pontos) dentro do HTML.
* **Dinamismo**: Criação e remoção de elementos (`createElement` e `remove`).
* **Estilização Dinâmica**: Controle de temas (Dark Mode) via JavaScript.
