# TaskDash 🚀

Sistema web focado em produtividade que une autenticação, utilitários de tempo e gamificação de tarefas.

## ✨ Funcionalidades
* **Acesso Controlado**: Sistema de login via JavaScript.
* **Relógio Dinâmico**: Monitoramento de tempo em tempo real (HH:MM:SS).
* **Customização**: Alternância entre Tema Claro e Tema Escuro (Dark Mode).
* **Lista Gamificada**: To-do list que soma e subtrai pontos conforme a conclusão das tarefas.

## 🛠 Tecnologias
* **HTML5 / CSS3** (Flexbox e Design Moderno)
* **JavaScript Vanilla** (Manipulação de DOM e Eventos)
