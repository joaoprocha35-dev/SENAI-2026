# ⚖️ Operadores Lógicos e Controle de Fluxo
Projetos focados em tomada de decisão e lógica booleana.

### 🧠 Conceitos Aplicados:
- **Operador `not`:** Inversão de valores lógicos.
- **Loops (`while` / `for`):** Criação de repetições inteligentes e controle de interrupção (`break`).
- **Ranges:** Geração de sequências numéricas para iterações.
