# 🐍 Lógica de Programação com Python

Repositório dedicado ao armazenamento de exercícios e projetos desenvolvidos durante o curso de **Fundamentos de Python** no SENAI. O foco aqui é o domínio da lógica algorítmica e estruturas base da linguagem.

## 🚀 Tecnologias e Ferramentas
* **Python 3.x**
* **VS Code** (Ambiente de Desenvolvimento)

## 🧠 Conceitos Praticados

Ao longo das aulas e módulos, os seguintes conceitos foram aplicados nos exercícios:

* **Fundamentos:** Variáveis, tipos de dados e entrada/saída de informações (`print`/`input`).
* **Estruturas Condicionais:** Tomada de decisão com `if`, `else` e `elif`.
* **Operadores Lógicos:** Uso avançado de `and`, `or` e `not`.
* **Estruturas de Repetição:** Controle de fluxos com laços `for` e `while`.
* **Matemática Aplicada:** Criação de calculadoras, cálculos de médias e operadores aritméticos.
* **Módulos e Bibliotecas:** Organização de código em múltiplos arquivos e importação de funções.

## 📂 Organização do Repositório

O projeto está dividido em pastas que acompanham a evolução do cronograma de estudos:

* `aula_03 / aula_04 / aula_05`: Exercícios progressivos de lógica.
* `Modulos-bibliotecas`: Práticas de organização e modularização de sistemas.
* `operador_not`: Desafios focados em lógica booleana e negação.
* `repetições`: Exercícios dedicados exclusivamente a loops e iterações.
