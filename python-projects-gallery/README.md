⚙️ Como Executar os Projetos
📥 Clone o repositório
git clone https://github.com/joaoprocha35-dev/python-projects-gallery.git
cd python-projects-gallery

📦 Instale as dependências

Recomendo usar ambiente virtual (venv)

pip install -r requirements.txt
🗄️ Configuração do Banco

Cada projeto possui suas próprias configurações em:

resources/database.py

Certifique-se de:

Ter o MySQL rodando
Criar as tabelas necessárias
▶️ Executar o projeto
python app.py

Acesse:

http://127.0.0.1:5000
