# 🎓 Sistema de Gestão de Alunos

Sistema CRUD desenvolvido com **Python**, **Flask** e **MySQL** para gerenciamento de alunos.

---

## 🚀 Tecnologias Utilizadas

- 🐍 Python 3
- 🌐 Flask
- 🛢️ MySQL
- 🎨 HTML5 & CSS3

---

## ✨ Funcionalidades

✅ Adicionar alunos  
✅ Listar alunos  
✅ Editar informações  
✅ Excluir registros  

---

## ⚙️ Como Executar

### 1️⃣ Instalar dependências

```bash
pip install flask mysql-connector-python
```

### 2️⃣ Executar o projeto

