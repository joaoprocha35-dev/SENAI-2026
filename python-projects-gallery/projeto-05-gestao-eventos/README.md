📅 SGE - Sistema de Gestão de Eventos

Sistema web desenvolvido em Flask + MySQL para gerenciamento de eventos com autenticação de usuários.

🚀 Funcionalidades
🔐 Login de usuários
👤 Registro de novos usuários
🚪 Logout seguro com sessão
📅 Cadastro de eventos
📋 Listagem de eventos ordenados por data
🧠 Sistema de sessão para proteção de rotas
🛠️ Tecnologias utilizadas
Python 3
Flask
MySQL
PyMySQL
HTML/CSS (frontend templates)
Jinja2 (templates Flask)
