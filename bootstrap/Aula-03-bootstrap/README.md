Componentes Práticos com Bootstrap 5
Este projeto foi desenvolvido como material de estudo para praticar a implementação de componentes dinâmicos e responsivos utilizando o framework Bootstrap 5. O foco foi entender a estrutura de classes utilitárias e o funcionamento do JavaScript nativo do framework.

🛠️ Tecnologias Utilizadas
HTML5: Estruturação semântica.

Bootstrap 5.3: Framework CSS para estilização e componentes.

Google Fonts/Icons.

🏗️ Principais Componentes Implementados
1. Barra de Navegação
Foi criada uma navbar responsiva com as seguintes características:

Menu Hamburger: Ativado automaticamente em dispositivos móveis através do navbar-toggler.

Dropdown: Menu suspenso para a seção de Portfólio.

Formulário de Busca: Integrado diretamente no alinhamento da barra de navegação.

2. Badges
Utilizamos badges para destacar as tecnologias dominadas:

Aplicação de bordas arredondadas com a classe rounded-pill.

Uso de cores contextuais como bg-danger, bg-warning e bg-info.

Exemplo de notificação dentro de um botão (badge de contador).

3. Accordion 
Implementação de um painel expansível para organizar informações de forma limpa:

Interatividade: Uso de data-bs-toggle="collapse" para abrir e fechar os itens.

Conteúdo Estruturado: Explicação detalhada sobre Java, HTML e CSS dentro dos colapsáveis.

4. Formulários
Criação de uma área de login e captura de dados contendo:

Controles de Formulário: Estilização de inputs de e-mail e senha com form-control.

Checkboxes: Opção de "Manter conectado" com alinhamento padrão do Bootstrap.

Área de Texto: Uso de textarea para comentários, configurado com linhas definidas.

5. Utilidades de Layout
Containers: Uso de container e container-fluid para controle de largura.

Espaçamento: Aplicação de margens (m-) e paddings (p-) para ajuste fino do layout.

Tipografia: Alinhamento de texto centralizado e cores personalizadas através de classes utilitárias.

✒️ Notas de Aula
O aprendizado focou na importância de importar corretamente o Bundle do Bootstrap (JS) ao final do body para que componentes como o Dropdown e o Accordion funcionem corretamente, além de entender o sistema de cores e botões.
